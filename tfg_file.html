Automation of an Industrial Pipe Assembly Process

Universitat Politècnica de Catalunya (UPC)

Supervisor: -

PI: Carlos Sierra Garriga

This document is based on the design of an automatic assembly module for parts, aimed at performing tasks that are both heavy and repetitive for operators. The process involves the assembly of brass clamps.

Five independent phases are defined, which together constitute the automated process. Prior to this, machinery and electronic components have been selected. Additionally, some custom elements have been manufactured, and specific mechanisms have been designed to ensure proper functionality. Using the SolidWorks CAD software for mechanical modeling, each stage of the process has been defined, followed by the assembly of the complete module as a final solution.

Leading to the following global solution:

Three out of the five phases of the automated process involve UR3 articulated robots. For this reason, virtual simulations were conducted using the URSim offline simulation software. Subsequently, these programs were transferred to the laboratory to subject the robot to a real working environment and verify its proper operation, enabling accurate calculation of cycle times.

Once the technical aspect of the project is completed, a plan for the implementation of this module is established, along with the preparation of a budget and an evaluation of the economic feasibility of the investment.
