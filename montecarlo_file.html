Where Am I? Monte Carlo Robot Localization

Universitat Politècnica de Catalunya (UPC)

In this project, we work with a mobile robot that needs to figure out its position on a known map, without using GPS. To do this, we use a method called Monte Carlo Localization, which simulates many possible robot positions (called particles). As the robot moves and scans its surroundings with a Lidar sensor, it gradually eliminates the less likely positions and focuses on the ones that make the most sense. We tested this technique on different maps, observing how the accuracy changes depending on the number of particles used, and how the robot “figures out” where it is as it moves around.
