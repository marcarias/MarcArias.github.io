Coding Conway's Game of Life in C

Universitat Politècnica de Catalunya (UPC)

This project consists of the implementation of Conway's Game of Life in the C programming language. The main objective is to simulate the evolution of cellular automata over time based on a set of simple rules. The implementation focuses not only on the game logic, but also on the efficient use of memory through dynamic allocation with malloc, proper use of pointers, and structured memory management. Several initial configurations are tested to observe different behaviors and patterns, including Diehard, R-pentomino, In-conf, and Acorn, each offering unique dynamics and complexity. This allows for analysis of both performance and memory handling in various scenarios.
