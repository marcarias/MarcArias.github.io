Real-Time 3D Head Pose Estimation from a Single Camera

Universitat Politècnica de Catalunya (UPC)

The main goal of this project was to estimate the 3D position and orientation of the head in real time using just a standard laptop webcam. Several methods were explored, all based on classic computer vision techniques to avoid training AI models and keep the system as lightweight and computationally efficient as possible.
